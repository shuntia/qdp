# qdp — Quake (and Disaster) Datagram Protocol

## JSON Communication Specification — v1.0 (draft)

**Author:** shuntia
**Status:** Draft / implementable

**Scope:**
This document specifies the JSON-based communication used by qdp clients to interact with qdp relays. This will cover hello, registration, information synchronization, and listening.

---

## 0. Design goals

- Simple, readable, and easy-to-implement JSON messages.

- Opt-in security features (encryption, authentication).

- Guaranteed message delivery with acknowledgments.
